Description:
Developed a full-stack web application, WanderLust, that allows users to list, search, and book vacation rentals such as cabins, beach houses, and more. The platform enables property owners to create and manage listings with essential details like price, location, and capacity, while travelers can explore, filter by categories (e.g., mountains, pools, castles), and securely book accommodations.

Key Features:

User Authentication: Implemented secure user authentication with Google OAuth and traditional email/password login.
Booking System: Integrated Stripe payment gateway for booking transactions with real-time session handling, check-in/out dates, and guest capacity validation.
Category-Based Search & Filters: Added filtering options based on property categories, allowing users to efficiently search for relevant listings.
Responsive UI: Designed an intuitive user interface with Bootstrap for a seamless experience across desktop and mobile devices.
Review System: Implemented a dynamic review and rating system for users to leave feedback on properties.
Admin Features: Included listing management, where owners can edit or delete their properties, as well as view booking history.
Dark Mode: Added a dark mode switch for enhanced user accessibility.
Tech Stack:

Frontend: EJS, Bootstrap, JavaScript, CSS
Backend: Node.js, Express.js, MongoDB, Mongoose, Multer
Payment Integration: Stripe API for secure payment handling
Deployment: Hosted on Render with MongoDB Atlas as the database service
